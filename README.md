# 🤖 ML Mini Tasks Repository

## 📌 Overview

This repository contains a collection of **Machine Learning mini projects and tasks**, focusing on practical implementation of core ML concepts. Each notebook demonstrates a specific problem, making it ideal for learning through hands-on experience.

The tasks range from **basic data analysis to advanced deep learning using transformer models like BERT**.
